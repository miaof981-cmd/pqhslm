## 🎨 新设计特点

### 1️⃣ 配色方案
- **主背景**：绿色渐变 `#A8E6CF → #74C69D`
- **按钮颜色**：白色背景 + 绿色文字 `#52B788`
- **文字颜色**：纯白色 + 半透明白色
- **阴影**：绿色系阴影，更柔和

### 2️⃣ 布局优化
```
┌─────────────────────────────────┐
│ 💰 张三                          │
│    视频专员                       │
├─────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐    │
│  │ ¥128.50  │  │ ¥200.00  │    │
│  │ 可提现    │  │累计已赚   │    │
│  └──────────┘  └──────────┘    │
├─────────────────────────────────┤
│        [立即提现]                │
└─────────────────────────────────┘
```

**只显示两个核心数据：**
- **可提现**：当前可以提现的金额（强调显示，更大字体）
- **累计已赚**：总分成收入（次要显示，较小字体）

### 3️⃣ 交互优化
- **立即提现按钮**：全宽按钮，点击动画
- **毛玻璃效果**：数据卡片使用半透明背景
- **响应式**：按钮点击有缩放反馈

## 🎯 显示逻辑

### 数据来源
```javascript
// 可提现 = 总分成 - 已提现
available = totalShare - withdrawn

// 累计已赚 = 总分成
totalShare = 所有订单分成的总和
```

### 显示条件
- ✅ 当前用户在"人员管理"中有记录
- ✅ "后台权限"开关开启
- ✅ `isActive = true`

---

## 💡 使用流程

### 用户视角
1. 进入管理后台，默认显示仪表盘
2. 顶部看到绿色收入卡片
3. 查看"可提现"和"累计已赚"
4. 点击"立即提现"按钮
5. 跳转到提现页面进行提现操作

### 视觉层级
```
最重要：可提现金额（大字体 + 更突出背景）
    ↓
次要：累计已赚（较小字体）
    ↓
操作：立即提现按钮（醒目的白色按钮）
```

## 🔧 自定义调整

### 修改颜色
编辑 `admin/index.wxss`，找到 `.my-income-card`：

```css
.my-income-card {
  /* 主背景渐变 */
  background: linear-gradient(135deg, #A8E6CF 0%, #74C69D 100%);
  
  /* 可改为其他绿色：
     浅绿：#C8E6C9, #A5D6A7
     深绿：#66BB6A, #4CAF50
  */
}

.withdraw-btn-bottom {
  background: white;
  color: #52B788;  /* 按钮文字颜色 */
}
```

### 调整字体大小
```css
/* 可提现金额 */
.income-stat-item .stat-value {
  font-size: 44rpx;  /* 改大改小 */
}

/* 累计已赚金额 */
.income-stat-item .stat-value-small {
  font-size: 32rpx;  /* 改大改小 */
}
```

### 修改按钮文字
编辑 `admin/index.wxml`：
```xml
<button class="withdraw-btn-bottom" bindtap="goToWithdraw">
  <text>立即提现</text>  <!-- 改为其他文字 -->
</button>
```

## 🎨 样式细节

### 毛玻璃效果
```css
backdrop-filter: blur(10rpx);  /* 模糊背景 */
```

### 渐变背景
```css
background: linear-gradient(135deg, #A8E6CF 0%, #74C69D 100%);
/* 135deg = 从左上到右下的对角线渐变 */
```

### 阴影效果
```css
box-shadow: 0 8rpx 24rpx rgba(168, 230, 207, 0.35);
/* rgba最后一位是透明度 */
```

## 📝 维护说明

### 数据更新
每次进入仪表盘或刷新时，会自动调用 `loadMyIncome()` 更新数据。

### 显示隐藏
只有当前用户是管理员时才显示，通过 `wx:if="{{myIncome.isStaff}}"` 控制。

### 样式修改
所有样式集中在 `admin/index.wxss` 末尾，搜索 `my-income-card` 即可找到。
